Projekt gra arkanoid. Gra polega na odbijaniu piłki za pomocą paletki i niszczeniem bloków, celem gry jest zebraniem jak największej ilości punktów.
Gra ma możliwość zapisania obecnych postępów i wczytywania zapisu gry. Jest też umeisczona tablica wyników zapisująca najlesze wyniki gracza.
Jako ruchome elementy zostały dodane gwiazdki w menu głównym oraz podczas gry które są elementem wizualnym oraz druga piłeczka która pojawia się po 
przekroczeniu progu 5 punktów
